# CNN & GNN Representation Learning

A theoretical exploration of representation learning in Convolutional Neural Networks (CNNs) and Graph Neural Networks (GNNs), focusing on receptive fields, residual connections, contrastive learning, and oversmoothing.

## Overview

This project investigates several advanced concepts used in modern deep learning architectures and analyzes how these techniques influence feature learning and model performance.

## Topics Covered

### Convolutional Neural Networks

* Theoretical vs Effective Receptive Field
* Kernel Size Effects
* Pooling and Subsampling
* Dilated Convolutions
* Deep CNN Architectures

### Graph Neural Networks

* Message Passing
* Feature Aggregation
* Oversmoothing
* Small-World Networks
* Deep GNN Challenges

### Representation Learning

* Residual Connections (ResNet)
* Contrastive Learning
* Feature Preservation
* Information Propagation

## Key Questions

* Why are large receptive fields important?
* How do residual connections mitigate oversmoothing?
* How does contrastive learning improve feature representations?
* Why do deep GNNs suffer from oversmoothing?

* Deep Learning Theory
